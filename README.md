# MERN JWT AUTH

Mern (MongoDB, Express, React, Node) project using JWT for email only authentication.

## Install

```
$ git clone git@github.com:DanielKanyo/mern-jwt-auth.git
$ cd mern-jwt-auth
$ npm i
```

## Development

You need a `.env` file with the following:
```
API_PORT=...
MONGO_URI=...
TOKEN_KEY=...
EMAIL_HOST=...
EMAIL_USER=...
EMAIL_PASSWORD=...
```

To start the sever:
```
$ npm run dev-server
```
To start the client:
```
npm run dev-client
```
